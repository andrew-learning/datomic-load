# datomic-load

Clojure project to try out Datomic Free.
Examples taken from http://markwatson.com/blog/2012-07/using-the-datomic-free-edition-in-a-lein-based-project.html

## Usage

sh transactor.sh &
lein repl

Run datomic-load.core in REPL
